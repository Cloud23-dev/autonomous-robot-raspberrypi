# Robot Autonome — Raspberry Pi

## Description
Développement d'un système de contrôle pour robot autonome sur Raspberry Pi.
Le robot implémente des algorithmes de navigation autonome (gradient ascent, exploration 
en spirale) et interagit avec son environnement via les interfaces GPIO.

## Fonctionnalités
- Contrôle moteur via GPIO
- Navigation autonome par gradient ascent
- Exploration systématique par algorithme en spirale
- Gestion des capteurs et périphériques

## Stack technique
- **Langage :** C / Python
- **Plateforme :** Raspberry Pi
- **Interface :** GPIO
- **OS :** Raspberry Pi OS
